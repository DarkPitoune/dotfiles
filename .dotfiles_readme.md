# Dotfiles

## Dependencies

### Shell
- [Oh My Zsh](https://ohmyz.sh/) - zsh framework
- [zoxide](https://github.com/ajeetdsouza/zoxide) - smarter cd
- [thefuck](https://github.com/nvbn/thefuck) - command correction

### Editors
- [Neovim](https://neovim.io/) - text editor
- Vim - fallback editor
- [vim-plug](https://github.com/junegunn/vim-plug) - Vim plugin manager

### Terminal
- [Tmux](https://github.com/tmux/tmux) - terminal multiplexer
- [TPM](https://github.com/tmux-plugins/tpm) - tmux plugin manager

### CLI tools
- [ripgrep](https://github.com/BurntSushi/ripgrep) - fast grep
- [fd](https://github.com/sharkdp/fd) - fast find
- [jq](https://jqlang.github.io/jq/) - JSON processor
- [Claude Code](https://github.com/anthropics/claude-code) - AI CLI

### Package managers
- [pnpm](https://pnpm.io/)
- [bun](https://bun.sh/)
- [nvm](https://github.com/nvm-sh/nvm) (macOS only)
- [Homebrew](https://brew.sh/) (macOS only)

### macOS only
- [Karabiner-Elements](https://karabiner-elements.pqrs.org/) - keyboard customization

## Setup

```bash
git clone --bare git@github.com:DarkPitoune/dotfiles.git ~/.dotfiles
alias dotfiles='git --git-dir=$HOME/.dotfiles --work-tree=$HOME'
dotfiles checkout
```

### Machine-specific config

Create these files for local overrides (gitignored):

- `~/.tmux.conf.local` — set `@host-name` and `@host-color` to identify the machine in the status bar
- `~/.zshrc.local` — machine-specific exports, aliases, and tool config

## Usage

```bash
dotfiles status
dotfiles add ~/.file
dotfiles commit -m "message"
dotfiles push
```
