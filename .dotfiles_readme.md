# Dotfiles

## Dependencies

Install these tools for the config to work:

### Shell
- [Oh My Zsh](https://ohmyz.sh/) - zsh framework
- [zoxide](https://github.com/ajeetdsouza/zoxide) - smarter cd
- [thefuck](https://github.com/nvbn/thefuck) - command correction

### Editors
- [Neovim](https://neovim.io/) - text editor
- Vim - fallback editor

### Terminal
- [Tmux](https://github.com/tmux/tmux) - terminal multiplexer

### macOS
- [Karabiner-Elements](https://karabiner-elements.pqrs.org/) - keyboard customization

### Package managers
- [pnpm](https://pnpm.io/)
- [bun](https://bun.sh/)
- [nvm](https://github.com/nvm-sh/nvm)
- [Homebrew](https://brew.sh/)

## Setup

```bash
git clone --bare git@github.com:USERNAME/dotfiles.git ~/.dotfiles
alias dotfiles='git --git-dir=$HOME/.dotfiles --work-tree=$HOME'
dotfiles checkout
```

## Usage

```bash
dotfiles status
dotfiles add ~/.file
dotfiles commit -m "message"
dotfiles push
```
